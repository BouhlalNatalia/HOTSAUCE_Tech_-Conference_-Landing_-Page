## 🇺🇸 English Description

A multi-section, fully responsive landing page (6 sections) developed as part of a Front-end course. This project showcases modern CSS layouts and custom interactive components.

### 🛠 Tech Stack
* **HTML5** — Semantic and accessible markup.
* **CSS3** — Flexbox, CSS Grid, Custom Properties, and smooth transitions.
* **JavaScript (ES6+)** — Accordion logic and Cookie notice management.
* **Responsive Design** — Custom breakpoints for a seamless experience: **1250px, 992px, 768px, 560px, 450px, and 390px**.

### 🌟 Key Features
* **Advanced Grid Layouts:** Used for the "Empathy" and "Reasons to attend" sections.
* **Interactive Elements:** A custom-built FAQ accordion and a functional Cookie consent bar.
* **Visual Polish:** Implemented "tear-off" ticket effects using CSS pseudo-elements (`::before`/`::after`).
